    Frequently Asked Questions

    What JSON Schema version does this generate?
    This tool generates JSON Schema Draft 7 (http://json-schema.org/draft-07/schema#). Draft 7 is widely supported by validators across all major programming languages and remains the most commonly used version in production.
    What is the difference between Strict, Balanced, and Loose modes?
    Strict mode marks every property as required. Balanced mode (default) infers required fields from presence in the input. Loose mode omits the required array entirely, making all properties optional. Choose based on how strict your validation needs to be.
    How does enum detection work?
    When an array contains a small number of primitive values (strings, numbers, booleans, or null), the tool extracts unique values and generates an enum constraint. The threshold is configurable — arrays with fewer unique items than the threshold produce enums instead of generic type descriptions.
